# ServiceNow Attachments Media

Enhances the default attachments component in ServiceNow by adding **inline video and audio playback**, so users can preview media files without downloading them.

---

## ✨ Features

* 🎬 Inline video playback (MP4, WebM)
* 🎧 Audio playback (MP3, WAV)
* 👀 Instant media preview inside attachments
* ⚡ No file download required
* 🔌 Seamless integration with existing attachments UI

---

## 🚀 Why this exists

Out of the box, ServiceNow attachments do not support media preview.
Users must download files to view or listen to them, which slows down workflows and hurts UX.

This component solves that by enabling **native media playback directly in the attachments list**.

---

## 🧩 Supported formats

| Type  | Formats   |
| ----- | --------- |
| Video | MP4, WebM |
| Audio | MP3, WAV  |

> Browser support may affect playback availability.

---

## 🛠️ Installation

1. Clone the repository
2. Import the component into your ServiceNow instance
